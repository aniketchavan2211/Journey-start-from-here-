## Adding Two Numbers

 - static [Fixed value]
 - dynamic [Take User input]

### static 
```js
let a = 5;
let b = 6;

var c = a + b;

console.log("The sum of "+a+" and "+b+" is "+c);
```

### dynamic
```js
let a = prompt("Please enter first number","");
let b = prompt("Please enter second number,"";

var c = parseInt(a) + parseInt(b);

console.log(" The sum of "+a+" and "+b+" is "+c);
```
